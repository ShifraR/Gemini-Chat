🤖 Gemini AI Chat (NetFree Friendly)
צ'אטבוט אינטראקטיבי המבוסס על מודל ה-AI של גוגל (Gemini), עם ממשק משתמש מודרני מבוסס Streamlit.
הפרויקט הותאם במיוחד לעבודה חלקה בעברית (RTL) ובסביבת האינטרנט של נטפרי.

✨ תכונות עיקריות
ממשק עברי מלא: יישור לימין (Right-to-Left) בבועות הצ'אט ובשדה הקלט.

ניהול זיכרון: הבוט זוכר את תוכן השיחה הקודם ומגיב בהתאם להקשר.

תאימות לנטפרי: הגדרה אוטומטית של תעודות אבטחה (SSL) והודעת חסימה מותאמת אישית.

🚀 איך מריצים את הפרויקט?
1. הכנת הסביבה (בטרמינל)
פתחו את תיקיית הפרויקט בטרמינל והריצו את הפקודות הבאות:

Bash
# יצירת סביבה וירטואלית
python -m venv venv

# הפעלת הסביבה (Windows)
.\venv\Scripts\activate

# התקנת הספריות הדרושות
pip install -r requirements.txt
2. הגדרת מפתח ה-API (API Key)
כדי שהצ'אט יוכל להתחבר למודל של גוגל, עליכם להגדיר את המפתח האישי שלכם:

בתיקיית הפרויקט תמצאו קובץ בשם .env.

פתחו אותו והחליפו את הטקסט your_api_key_here במפתח שקיבלתם מגוגל.

שמרו את הקובץ.

3. הרצת האפליקציה
כדי להפעיל את ממשק הצ'אט בדפדפן, הריצו בטרמינל את הפקודה:

Bash
streamlit run web_app.py
הדפדפן ייפתח אוטומטית בכתובת http://localhost:8501.

🔑 איך מקבלים מפתח API? (מדריך למתחילים)
היכנסו לאתר Google AI Studio.

התחברו עם חשבון הגוגל (Gmail) שלכם.

בתפריט הצדדי, לחצו על "Get API key".

לחצו על "Create API key in new project".

העתיקו את הקוד שנוצר והדביקו אותו בקובץ ה-.env.

🛡️ הערות למשתמשי נטפרי
הפרויקט מותאם במיוחד לעבודה חלקה בסביבת נטפרי:

SSL: הקוד מחפש אוטומטית את התעודה בנתיב C:\ProgramData\NetFree\CA\netfree-ca-bundle-curl.crt.
