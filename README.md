# 🤖 Gemini AI Chat (NetFree Friendly)

צ'אטבוט המבוסס על Gemini API עם ממשק Streamlit מותאם לעברית ולנטפרי.

## 🛠️ איך מריצים?
1. צרו סביבה וירטואלית: `python -m venv venv`
2. הפעילו אותה: `.\venv\Scripts\activate`
3. התקינו ספריות: `pip install -r requirements.txt`
4. צרו קובץ `.env` עם ה-API Key שלכם.
5. הריצו: `streamlit run web_app.py`
